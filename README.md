## Project Description

HabitFlow is a Python application that tracks daily habits and analyzes progress over time. The application stores habit data in a JSON file, calculates streaks and completion rates using datetime, and visualizes statistics using Matplotlib charts.

## Project Files

* `create_sample_data.py` — generates sample habit data.
* `habit_tracker.py` — habit management and statistics calculations.
* `stats_reporter.py` — reporting and JSON export.
* `visualizer.py` — charts and visualizations.
* `main.py` — application entry point.

## How to Run

```bash
pip install -r requirements.txt
python create_sample_data.py
python main.py
```
